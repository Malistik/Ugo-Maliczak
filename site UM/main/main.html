Pourquoi avoir fait ce site ?

Ce site est un projet pour la matière NSI qui vise à enseigner les fondamentaux de l'informatique comme:

  • Traitement des données sous la forme numérique: textes, images, sons...
  • L'algorithmique: Méthode de calculs sur les données.
  • Les langages: Traduction des algorithmes pour qu'ils soient éxécutables par des machines.
  • Les machines: Systèmes d'exploitations, réseaux, stockage des données
